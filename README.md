# 💪 EASY MUSCLE

App de rutinas de hipertrofia personalizada con IA.

## Deploy en Vercel (5 minutos)

1. Subí esta carpeta a GitHub
2. Entrá a vercel.com → New Project
3. Importá el repo → Deploy
4. ¡Listo! Compartí el link con tus clientes.

## Desarrollo local

```bash
npm install
npm run dev
```
